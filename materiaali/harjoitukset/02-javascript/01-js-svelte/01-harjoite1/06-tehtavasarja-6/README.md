# Tehtäväsarja 6 - komponentit kuntoon

Tässä tehtäväsarjassa määritämme minkälaista skalaaridataa minkäkin komponentin tulisi ottaa vastaan.

Siltä osin kun se on yksinkertaista ja helppoa, käytämme myös tätä dataa informaation näyttämiseen.

Tämä on kuitenkin vasta ensimmäinen kierros datan osalta, ja palaamme myöhemmässä tehtäväsarjassa määrittämään kuntoon myös ne komponentit jotka käyttävät monimutkaisempaa dataa.

Nyt kuitenkin siis vielä keskitymme yksinkertaisen _skalaarisen_ datan käyttöönottoon.

_Skalaaridata tarkoittaen yksittäisiä tekstejä ja numeroita._

Taulukko- ja objektimuotoisen datan lisäämme myöhemmässä tehtäväsarjassa.
