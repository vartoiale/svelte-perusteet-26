# Tehtäväsarja 3

Nyt kun olemme lisänneet näkyvää, 
mutta tilapäistä sisältöä kaikkiin komponentteihin,
on aika liittää nämä komponentit yhteiseksi kokonaisuudeksi: komponenttipuuksi.

Komponenttipuu tarkoittaa yhdestä tai useammasta komponentista rakentuvaa puuta, 
jossa komponentit linkittyvät puumaiseksi rakenteeksi.

Teemme seuraavaksi aiemmin käsitellyistä komponenteista puun.
Teemme tämän kansio kerrallaan kansioille välillä `teht07` - `teht27`.

Näissä tehtävissä on tarkoituksena vain:

* käyttää `import`-syntaksia alikomponenttien lukemiseen,
* näyttää alikomponentit graafien mukaisesti.

Älä siis vielä lähde toteuttamaan muutoin komponenttien sisältöä.
Se tehdään myöhemmässä vaiheessa.
