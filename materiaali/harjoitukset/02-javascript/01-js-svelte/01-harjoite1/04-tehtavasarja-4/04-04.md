# Alapalkin sarakkeiden alustava tyylittely

Kun `AlapalkinSarake.svelte`-komponentti on nyt käytössä `teht23`-kansion komponenteissa,
voimme seuraavaksi tyylitellä niiden vanhempana toimivat komponentit:

* `alapalkin-alasarakkeet.svelte`
* `alapalkin-ylasarakkeet.svelte`

## Näytettävät sarakkeet

`AlapalkinYlasarakkeet`-komponentin pitäisi näyttää seuraavat sarakkeet:

* `AsiakaspalveluYritysmyynti`
* `Myymalat`
* `Yhteisot`
* `Uutiskirje`

`AlapalkinAlasarakkeet`-komponentin pitäisi näyttää seuraavat sarakkeet:

* `Info`
* `Shop-in-shops`
* `Artikkelit`
* `Palkinnot`

## Toteutus

Nyt `AlapalkinYlasarakkeet`-komponentin pitäisi sisältää seuraava html-osio:

_`alapalkin-ylasarakkeet.svelte`:_

```svelte
<div>
    <AsiakaspalveluYritysmyynti />
    <Myymalat />
    <Yhteisot />
    <Uutiskirje />
</div>
```

Vastaavasti `AlapalkinAlasarakkeet`-komponentin pitäisi sisältää seuraava html-osio:

_`alapalkin-alasarakeet.svelte`:_

```svelte
<div>
    <Info />
    <ShopInShops />
    <Artikkelit />
    <Palkinnot />
</div>
```

Lisäämme näihin seuraavaksi yksinkertaiset tyylit,
joilla määritämme ne lomittumaan horisontaalisesti flexbox:in avulla:

_`alapalkin-ylasarakkeet.svelte`:_

```svelte
<div>
    <AsiakaspalveluYritysmyynti />
    <Myymalat />
    <Yhteisot />
    <Uutiskirje />
</div>

<style>
    div {
        display: flex;
        flex-direction: row;
    }    
</style>
```

ja 

_`alapalkin-alasarakeet.svelte`:_

```svelte
<div>
    <Info />
    <ShopInShops />
    <Artikkelit />
    <Palkinnot />
</div>

<style>
    div {
        display: flex;
        flex-direction: row;
    }    
</style>
```

Nyt alapalkin sarakkeiden pitäisi jo vähitellen näyttää oikeammalta.

Voit nyt tyylitellä näitä sarakkeita halutessasi enemmänkin.

## Seuraavaksi

Seuraavaksi tutustumme siihen,
miten svelte:ssä voidaan tyylitellä komponentteja yli komponentin rajojen.
