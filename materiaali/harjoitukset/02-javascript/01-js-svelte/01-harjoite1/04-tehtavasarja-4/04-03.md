# Otetaan käyttöön `teht22/alapalkin-sarake.svelte`- ja `teht21/alapalkin-osio.svelte`-komponentit

Muokkaa nyt `teht23`-kansion komponentit käyttämään `teht22/alapalkin-sarake.svelte`- ja `teht21/alapalkin-osio.svelte`-komponentteja niiden uudella toteutuksella.

Kun aiemmin `teht23`-kansion komponenteissa käytit näitä kahta komponenttia itsensäsulkevina komponentteina:

`teht23/info.svelte`, aiemmin:

```svelte
<script>
    import InfoSivut from "../teht18/info-sivut.svelte";
    import AlapalkinOsio from "../teht21/alapalkin-osio.svelte";
    import AlapalkinSarake from "../teht22/alapalkin-sarake.svelte";
</script>

<div>info</div>

<AlapalkinSarake />
<AlapalkinOsio />
<InfoSivut />
```

Nyt haluammekin laittaa aiemman `teht23/artikkelit.svelte`-esimerkin lailla nämä komponentit toimimaan kuin `div`-elementit.

`teht23/info.svelte`, muokataan seuraavaksi:

```svelte
<script>
    import InfoSivut from "../teht18/info-sivut.svelte";
    import AlapalkinOsio from "../teht21/alapalkin-osio.svelte";
    import AlapalkinSarake from "../teht22/alapalkin-sarake.svelte";
</script>

<div>info</div>

<AlapalkinSarake>
  <AlapalkinOsio otsikko="Info">
    <InfoSivut />
  </AlapalkinOsio>
</AlapalkinSarake>
```

Tee siis tämä sama muutos myös muihin `teht23`-kansion komponentteihin, 
joissa näitä kahta komponenttia käytetään.

Muista antaa `AlapalkinOsio`-komponentille `otsikko`-attribuuttina referenssin mukainen otsikko,
kuten esimerkissä yllä.
