#### Komponentti: `projekti-pudotusvalikko-nappi.svelte`:

Tee nappi, jonka asetat projekti-headeriin. Nappia painamalla pudotusvalikon tulee avautua.

**palautettavien tiedostojen ja kansioiden nimet:** 

* tiedosto: `teht40/projekti-pudotusvalikko.svelte` (kansiossa: `harjoitukset/02-javascript/01-svelte/teht40/projekti-pudotusvalikko-nappi.svelte`)

Voit käyttää napissa haluamaasi ikonia. 
Hampurilaismenu ( &#8801; ) on kuitenkin hyvä valinta tähänkin.

Voit halutessasi jättää napin tekemättä, jos keksit muun tavan avata pudotusvalikon.

## Esimerkkitoteutus

Esimerkkitoteutuksessa jatketaan edellisen sivun toteutusta.

Tässä kontekstista otetaan `"projekti-auki"`-nimellä edellisen sivun komponentissa sinne laitettu arvo,
ja se talletetaan `tila`-muuttujaan.

Napin painallukseen reagoimista varten luodaan `handleClick`-funktio,
joka kääntää `tila.auki`-arvossa olevan boolean arvon käänteiseksi napin painalluksella
(`true` -> `false`, tai `false` -> `true`).
Tämän toiminnasta ei tarvitse tässä yhteydessä täysin välittää.

Lopuksi `handleClick`-funktio annetaan buttonille käyttäen svelten `on:click`-riimua.
`on:click`-riimu reagoi `button`-elementtiin kohdistuvaan painallukseen.

_`teht40/projekti-pudotusvalikko-nappi.svelte`:_

```svelte
<script>
    import {getContext} from 'svelte';
    let tila = getContext("projekti-auki")
    let handleClick = (event) => {tila.auki = !tila.auki}
</script>

<button on:click={handleClick}>&#8801;</button>
```
