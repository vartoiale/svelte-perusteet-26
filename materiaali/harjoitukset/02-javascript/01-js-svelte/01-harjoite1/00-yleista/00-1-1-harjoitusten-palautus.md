# Harjoitusten kansio

Harjoitukset tehdään kansioon `/harjoitukset/02-javascript/01-svelte`.

## Harjoitusrepositorio

Kurssia varten on tehty valmiiksi konfiguroitu harjoitusrepositorio. Se on tätä kirjoittaessa avaimet käteen -kuntoinen, tarkoittaen, että se sisältää valmiin svelte-projektin, jossa on valmiina storybook, ja tarvittavat `.svelte`-komponenttitiedostot.

Se mitä versiota tästä harjoitusrepositoriosta käytät, riippuu siitä, teetkö tätä kurssia koulussa vai itsenäisesti.

### Harjoitusrepositorio: kevät 2026 osana ohjelmointi-tutkinnonosaa

Moodlessa on linkki github classroomiin, jonne on valmiiksi luotu ympäristö tehtäville.

### Harjoitusrepositorio: itsenäinen suoritus

Jos teet tätä harjoituskokonaisuutta kurssin ulkopuolella, voit luoda itsellesi oman harjoitusrepositorion, käyttäen apuna [vartoiale/svelte-perusteet-26-aloitus](https://github.com/vartoiale/svelte-perusteet-26-aloitus) repositoriota. Kyseinen repositorio on ns. sapluunarepositorio (engl. template repository), tarkoittaen, että sen pohjalta voi tehdä itselleen uuden repositorion. Valitsemalla sapluunaksi tuon repositorion, github alustaa uuden repositoriosi sapluunan (`vartoiale/svelte-perusteet-26-aloitus`) sen hetkisellä tiedostorakenteella.

#### Uuden repositorion teko sapluunarepositorion pohjalta

Käytä [vartoiale/svelte-perusteet-26-aloitus](https://github.com/vartoiale/svelte-perusteet-26-aloitus) sapluunana omalle uudelle repolle:

1. kirjaudu githubiin,
2. mene sapluunarepositorioon ([vartoiale/svelte-perusteet-26-aloitus](https://github.com/vartoiale/svelte-perusteet-26-aloitus)),
3. klikkaa oikeasta yläreunasta vihreätä nappia "use this template" > "create new repository",
4. täytä avautuvalla "New repository"-sivulla uuden repon tiedot.

## Harjoituskansion sijainti harjoitusrepositoriossa

Tehtävät löytyvät harjoitusrepositoriosta `/harjoitukset/02-javascript/01-svelte` -kansiosta.

Koska tiedostot ovat jo etukäteen luotu, on tärkeää, että teet tehtävät juuri niihin. Tällöin etukäteen konfiguroitu storybook osaa näyttää harjoituksiin tekemäsi muutokset oikein.

### Harjoitusten palautus

Jos teet harjoituksen github classroom:in kautta osana kurssia, harjoitukset pitää vielä palauttaa opettajan tarkastettavaksi.
Harjoitusten palautus tapahtuu commit:oimalla muutokset versionhallintaan, ja synkronoimalla muutokset palvelimelle.

Tällä kertaa tiedostot on luotu jo valmiiksi `harjoitus`-kansion alle, 
ja niihin tarvitsee vain lisätä oikeat ratkaisut.

Tarkasta siis tehtävän yhteydessä, mitä tiedostoa tehtävässä on tarkoitus muokata.
