# Tehtäväsarja 2: komponenttien alustus

Alkuun lisäämme jokaiseen komponenttiin sisältöä,
jotta tiedämme, mikä komponenteista näkyy,
kun katselemme tarinoita.

Tehtävissä vaaditut komponentit on luotu jo valmiiksi tyhjinä tiedostoina.
Tässä tehtäväsarjassa lisäämme jokaiseen komponenttiin tilanvaraaja sisältönä komponentin nimen.

Tästä kuitenkin varsinaisessa tehtäväkuvauksessa.
