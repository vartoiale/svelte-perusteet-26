## Uuden svelte-kit -projektin luonti

Uuden svelte-kit -projektin luonti onnistuu:

1. siirtymällä `/projektit`-kansioon:
   
   Aja repositorion juuressa:
   
   ```sh
   cd projektit
   ```

2. luomalla uuden projektin:
   
   Aja komento (korvaa `<projektin nimi>` haluamallasi projektin nimellä):

   ```sh
   yarn dlx sv create <projektin nimi>
   ```

3. siirtymällä uuteen projektiin:
   
   Aja komento:

   ```
   cd <projektin nimi>
   ```

Hyvä nimi projektille on muotoa: `svelte-<projekti>`.
Jos esimerkiksi teet frontendpractice.com-sivuston `abstract`-tehtävän, 
nimi olisi `svelte-abstract`.
